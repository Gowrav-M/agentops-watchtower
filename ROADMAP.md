# Roadmap

AgentOps Watchtower is now focused on adoption, runtime control, and broader agent coverage.

## Near Term

- Packaged GitHub release flow with signed release artifacts.
- More agent transcript adapters: Codex, Claude Code, Cursor, OpenCode, OpenHands, and Gemini CLI.
- Better PR summaries from AgentBOM, admission, runtime graph, and SARIF results.

## Next Breakthrough

- MCP protocol proxy and wrapper mode.
- Runtime allow/deny policy before tool execution.
- Per-tool approval prompts and signed approval evidence.

## Later

- SQLite storage for larger local histories.
- Local browser report viewer.
- Optional OPA/Rego-style policy packs.
- Organization-level MCP trust registry.
